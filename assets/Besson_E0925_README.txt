Besson E0925 Clarinet Model - OpenWind Starter Pack
---------------------------------------------------
This pack contains:
1. Besson_E0925_tomo_geometry.ow - OpenWind-ready internal bore geometry
2. Besson_E0925_impedance_target.txt - Measured impedance spectrum at 20°C
3. This README for guidance

USAGE:
- Load the .ow file using OpenWind’s geometry loader
- Use the impedance file to visualize or match via inverse modeling
- Recommended to use the TMM method for simulations

SOURCE:
Tomography and measurements by Elsa Lambert for C2RMF lab, data available via HAL:
https://inria.hal.science/hal-04352706
